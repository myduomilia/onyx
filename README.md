# ONYX

Micro framework for developing web applications

## Supported OS
Linux

## Install

### Required libraries

```bash
libfcgi-dev
libboost-all-dev
libcurl4-openssl-dev
```

```bash
make clean
make
make install
```
## Using:
* Logging  - https://github.com/SergiusTheBest/plog
* JSON - https://github.com/nlohmann/json
* Rendering templates - https://github.com/kainjow/Mustache
